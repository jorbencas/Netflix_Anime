* React
* React Native

### Configuración del proyecto

* Instalar node y npm en linux 
    -   sudo apt-get install wget
    -   wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
    -   source ~/.profile
    -   nvm ls-remote
    -   nvm install 12.18.4
    -   npm -v
    -   node -v
    
* Desinstalar nvm 
    -   nvm deactivate
    -   nvm uninstall v12.18.4
